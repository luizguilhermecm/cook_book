Utils
=====

#### Wrap existing text

`gqgq` Will wrap the line where cursor is.

If text is selected `gq`.


#### Files


| cmd  | function                  |
| ---- | ------------------------- |
| `gf` | Open file in same windown |
| `<c-w>f` | open in new horizontal window |
| `<c-w>gf` | open in tab |
| `<c-6>` | go to file before `gf` |
